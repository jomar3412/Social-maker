# Gemini CLI

This document outlines the usage and capabilities of the Gemini CLI for the socal_maker project.

## Overview

The Gemini CLI provides an interactive interface for managing and automating various tasks within the socal_maker project. It leverages the Gemini API to generate content, analyze data, and streamline video production workflows.

## Key Features

- **Content Generation:** Generate scripts, video ideas, and marketing copy using Gemini's language models.
- **Asset Management:** Integrate with Google Cloud Vision for asset tagging and organization.
- **Video Production Automation:** Automate tasks such as video assembly, subtitle generation, and kinetic text animation.
- **Project Configuration:** Easily configure project settings and credentials.

## Usage

To start the Gemini CLI, navigate to the project root directory and run:

```bash
python cli.py
```
